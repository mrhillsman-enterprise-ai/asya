Asya🎭

Kubernetes-native Async Actor Framework

Orchestrate AI/ML workloads at scale with a powerful actor-based framework built for Kubernetes. Deploy distributed systems with ease using familiar patterns.

Documentation GitHub Quick Start

Get Started

Add the Helm repository:

helm repo add asya https://asya.sh/charts

Install the operator:

helm install asya-operator asya/asya-operator -n asya-system --create-namespace
